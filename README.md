# DevOps Automation Engineering Tools

Prerequisite
- AWS Account
- Git Setup
- Ansible Setup

Installing Ansible on Windows

(1) Install cygwin from https://cygwin.com/install.html

(2) Install the following packages:

        curl
        python
        python-setuptools
        python-crypto
        openssl
        openssl-devel
        libffi-devel
        gcc-g++
        vim
        git

(3) Install pip

        easy_install-2.7 pip

(4) Install Ansible

        pip install ansible
